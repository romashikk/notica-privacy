Privacy Policy

This Privacy Policy applies to the Notica application (hereinafter referred to as the "Application") for mobile devices, which was created by Kawori (hereinafter referred to as the "Service Provider") as a free service. This service is provided "AS IS".

Data Collection and Storage

The Notica application is designed with maximum user data privacy in mind. All data (including tasks, notifications, categories, and settings) is stored exclusively on your device and is not transmitted to any servers. You have full control over your data through the backup feature, and all data will be deleted when the application is uninstalled.

App Permissions

The application requests the following permissions:

  • Notification permission - used for sending task reminders and notifications (all notifications are created and managed locally on the device).
  • Storage permission - used only for creating and restoring backups of your data (all backups are stored in your device's private application directory).

Data Backup

The application provides backup functionality with the following features:

  • All backups are stored locally on your device
  • Backup files contain only your tasks, categories, and application settings
  • No personal information is collected or transmitted during the backup process
  • You can manually delete all backup files through the application settings

Language and Localization

The application supports multiple languages (English and Russian). Your language preference is stored locally on your device and is not shared with any third parties.
